# `v0.50.0` 验收清单

## 说明

本清单是 `v0.50.0` 当前唯一有效的手工验收基线。

## 验收项

### 1. 主工作台新视觉语言

- 主工作台整体视觉统一为深色科技控制台风格。
- 主色基调保持深海军蓝到近黑底色，强调色以青蓝、电青为主，不回退到浅色常规待办风格。
- 顶部动作区、左侧工作视图和中间任务列表的层级关系保持清晰。
- 弹窗式概览、提醒、设置、任务详情和任务组管理沿用同一套面板语言。
- 优先级、截止时间、任务组和提醒状态在列表与弹窗中保持统一的状态芯片表达。
- 主文本仍保持高可读性，颜色强调仅用于状态与行动引导。

### 2. 顶部系统图标入口

- 顶部右侧存在独立的 `设置`、`提醒` 和 `备份` 图标入口。
- `提醒` 入口可展示待处理数量角标。
- `备份` 入口可独立进入备份与恢复中心，不依赖设置子项。
- 入口行为与文案保持一致，不混用其他版本的旧口径。

### 3. 提醒角标与提醒入口

- 提醒角标会随待处理提醒数量变化。
- 待处理数量为 0 时，角标不应显示为误导性的非零状态。
- 点击提醒入口后可以进入提醒详情或提醒中心，并保持当前工作台语义一致。

### 4. 备份与恢复闭环

- 可以手动创建备份。
- 可以从备份文件恢复数据。
- 备份与恢复面板能展示最近备份时间与基础状态信息。
- 恢复前会进行风险确认。
- 恢复失败不能打断当前应用数据读取链路。
- 恢复成功后会统一刷新任务状态。
- 非法文件、读取失败或格式不兼容时，会给出统一错误反馈。

### 5. 数据导出能力

- 可以导出 JSON。
- 可以导出 CSV。
- 全量导出在备份中心。
- 当前筛选结果导出在列表工具条。
- 导出流程不会误导为备份或恢复操作。

### 6. 全局搜索与快速定位

- 搜索入口在顶部主控制区。
- 可以按标题搜索任务。
- 可以按备注搜索任务。
- 搜索结果需要同时体现任务标题、状态、优先级和必要时间信息。
- 点击搜索结果后，会自动切换必要筛选条件。
- 目标任务会自动滚动到可视区域并短暂高亮。
- 搜索结果面板应保持与主工作台一致的视觉语言。
