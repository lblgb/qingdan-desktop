# 第一版验收清单

## 文件说明

本文档用于沉淀第一版在当前阶段的手工验收口径，作为每轮收口、发布前复核和问题回归的统一清单。

## 验收范围

- 桌面应用可启动并显示主窗口。
- 任务新增、编辑、完成、删除链路可正常工作。
- 数据在应用重启后仍能保留。
- 当前排序、分组、筛选与反馈语义保持稳定。
- 发布配置中的产品名、窗口标题和图标引用已显式配置。
